# OpenManus

Open-source general agent framework (Python, MIT) by the MetaGPT team (FoundationAgents org); launched March 2025 as a 3-hour prototype answer to the invite-only Manus agent. ReAct loop, tool collection (browser-use, Python exec, search), PlanningFlow multi-step execution, MCP client + server entry points. 56k+ stars, but semi-dormant since ~April 2025 (last release v0.3.0; ~343 open issues) (Source: [[openmanus-repo]]).

**Relevance to claude-mem**: rejected as a runtime (would require porting Agent Skills to Python); adopted as a pattern source for [[Plan-Driven Research Loop]]; potential future *consumer* of a claude-mem vault MCP server. Decision record: [[Research OpenManus for claude-mem]].
